# 💰 Expense Tracker Web App

A simple and interactive Expense Tracker built using **HTML, CSS, and JavaScript**.  
This project helps users manage daily expenses, categorize them, and track total spending in real-time with data persistence using localStorage.

---

## 🚀 Features

- ➕ Add new expenses with name, amount, and category
- 📂 Categorize expenses (Food, Travel, Shopping, etc.)
- 🧮 Automatic total expense calculation
- 🗑️ Delete individual expenses
- 💾 Data persistence using localStorage
- 🔄 Auto-load saved data on page refresh
- 🎯 Clean and responsive UI design

---

## 🛠️ Tech Stack

- HTML5 – Structure of the app
- CSS3 – Styling and layout
- JavaScript (ES6) – Logic and functionality
- localStorage – Data storage in browser

---

## ⚙️ How It Works

1. User enters expense name, amount, and category  
2. Data is stored in an array (`expenses`)  
3. Data is saved in `localStorage`  
4. UI updates dynamically using DOM manipulation  
5. Total expense is calculated automatically  
6. Data remains even after page refresh
